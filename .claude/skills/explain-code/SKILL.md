---
name: explain-code
description: Explains how the code works using diagrams and analogies.
---


When you explain the code, please include the below.

1. **Start analogies**: Illustrate the code as something in daily life.
2. **Draw diagrams**: Show its relation, structure and flow as an ASCII art.
3. **Walkthrough the code**: Explain step by step what happens.
4. **Enphasize notes**: What are some common mistakes and misconceptions?

Explaining should be tone like conversations. If you explain a complex conception, use multiple diagrams or analogies.
